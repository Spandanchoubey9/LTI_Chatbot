# Chatbot

A chatbot built using React, Node.js, and PostgreSQL.

# Features
- Predefined chatbot responses
- Chat history stored in PostgreSQL
- React-based chat interface
- Node.js + Express backend

# Tech Stack
- Frontend: React
- Backend: Node.js, Express
- Database: PostgreSQL

# How to Run

# Backend

cd chatbot-server
npm install
npm start

# Frontend

cd chatbot-frontend
npm install
npm start
